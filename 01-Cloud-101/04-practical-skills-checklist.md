# 🛠️ AWS Core Services: Practical Skills Checklist

**Date:** 21 Maret 2026
**Topic:** AWS Management Console Demos

Berikut adalah daftar kemampuan operasional dasar (*hands-on skills*) yang sudah saya pelajari cara eksekusinya melalui AWS Management Console:

## 📦 Storage & Compute
- [x] **Amazon S3:** Membuat *bucket*, mengunggah *object* (file), dan mengatur akses agar *object* bisa dibagikan (publik/privat).
- [x] **Amazon EC2:** Melakukan *provisioning* dan meluncurkan *virtual machine* (instans EC2) baru.

## 🗄️ Database & Networking
- [x] **Amazon RDS:** Meluncurkan instans *relational database* terkelola dan mengonfigurasi koneksinya ke aplikasi web.
- [x] **Amazon VPC:** Merancang dan men-*deploy* jaringan *Virtual Private Cloud*, mengatur komponen *default*-nya, dan meluncurkan EC2 tepat di dalam VPC tersebut.

## 🔐 Security & Serverless
- [x] **AWS IAM:** Membuat *User* dan *Group* baru, serta menempelkan *Policies* (hak akses) terkelola kepada *user* tersebut.
- [x] **AWS Lambda:** Membuat *serverless function* menggunakan *blueprint* (cetak biru), menguji *code*, dan memanggil fungsi tersebut.

## 📈 Monitoring
- [x] **Amazon CloudWatch:** Memantau metrik layanan, membuat *Alarm* peringatan, dan menghubungkannya dengan notifikasi (Amazon SNS).
