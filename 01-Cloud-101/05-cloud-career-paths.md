## 🎓 AWS Certification Roadmap
Untuk mencapai karir yang saya targetkan, berikut adalah *roadmap* sertifikasi AWS yang menjadi panduan saya:

1. **Foundational:** Lulus **AWS Certified Cloud Practitioner (CCP)** untuk memvalidasi pemahaman fundamental *cloud*.
2. **Associate:** Mengambil AWS Certified Solutions Architect sebagai standar keahlian teknis operasional.
3. **Specialty (Long-term goal):** Mendalami keahlian spesifik di bidang Securityuntuk meningkatkan nilai tawar di industri.
