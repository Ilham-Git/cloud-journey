# ☁️ Module 01: Cloud Computing 101

**Date Started:** 16 Maret 2026
**Platform:** AWS Educate - Cloud 101

## 🎯 Course Objective
Modul pengantar ini dirancang untuk membangun fondasi pemahaman terkait *Cloud Computing*. Tujuannya adalah memberikan gambaran besar (*high-level overview*) sebelum terjun ke materi teknis yang lebih spesifik, serta membantu menentukan arah *learning path* selanjutnya.

## 🚀 Learning Goals
Setelah menyelesaikan modul ini, target penguasaan *skill* saya meliputi:

* **Fundamental Concepts:** Mendefinisikan apa itu *Cloud Computing* dan menjelaskan *benefits* (manfaat) utamanya bagi bisnis dan operasional IT.
* **Architecture Models:** Membandingkan berbagai *Cloud Computing Service Models* (*IaaS, PaaS, SaaS*) beserta *Deployment Models* (*Public, Private, Hybrid Cloud*).
* **AWS Ecosystem:** Mengidentifikasi komponen penyusun *AWS Global Infrastructure* (*Regions* dan *Availability Zones*).
* **Security:** Memahami *Shared Responsibility Model* terkait pembagian tugas keamanan antara AWS dan *customer*.
* **Best Practices:** Menjelaskan *AWS Well-Architected Framework* dan perannya dalam membangun arsitektur sistem yang fleksibel dan *reliable* (andal).
* **Core Services:** Mengenali fungsi, fitur, dan kegunaan dari layanan utama AWS (*AWS Core Services*).
* **Hands-on Experience:** Melakukan praktik langsung di dalam *Lab Environment* untuk mencoba *core services* tersebut.
* **Career Path:** Memetakan empat jalur karir *entry-level* di ekosistem *Cloud Computing*.

> **📌 Important Note:**
> *Course* ini bersifat interaktif dan menggabungkan format audio serta teks.
