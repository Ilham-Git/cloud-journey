# ☁️ My Cloud Engineering Journey

Hi! Saya Ilham. *Repository* ini adalah ruang dokumentasi personal saya dalam mempelajari *Cloud Computing* dari nol hingga siap terjun sebagai seorang *Cloud Engineer* profesional. 

Semua catatan di sini menggunakan pendekatan *hybrid* (Bahasa Indonesia dengan *English technical terms*) untuk mempercepat pemahaman konsep sekaligus membiasakan diri dengan standar industri global.

## 🎯 Goals
- Memahami fundamental arsitektur *Cloud Infrastructure* (khususnya AWS).
- Lulus ujian sertifikasi **AWS Certified Cloud Practitioner**.
- Membangun *hands-on portfolio* seputar *provisioning*, *networking*, *security*, dan *monitoring*.
